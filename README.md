# COMP3011_cw2

## Project Overview

This project implements a small‑scale web search engine for the website  
**<https://quotes.toscrape.com>**, developed as part of *COMP3011 – Web Services and Web Data*.

The tool demonstrates the core components of a search engine:

*   **Web crawling** with politeness constraints
*   **Inverted index construction** with term statistics
*   **Information retrieval** using TF‑IDF ranking
*   **Boolean AND** and **phrase queries**
*   A **command‑line interface (CLI)** for user interaction

The system is designed for clarity, correctness, and robustness rather than large‑scale performance, reflecting the educational focus of the assignment.

***

## Purpose and Learning Objectives

The project aims to demonstrate:

*   Understanding of **web crawling techniques** and polite scraping practices
*   Implementation of an **inverted index** with frequency and positional data
*   Application of **TF‑IDF ranking** for search relevance
*   Correct handling of **multi‑word, phrase, and edge‑case queries**
*   Use of **unit testing and version control** to support reliable development

## Command‑Line Usage

All functionality is accessed via `main.py`.

### 1. Build the index

Crawls the website, builds the inverted index, and saves it to disk.

```bash
python src/main.py build
```

Key features:

*   Breadth‑first crawl
*   Pagination‑only traversal (`/page/<n>`)
*   6‑second politeness window between requests
*   Graceful handling of network errors

***

### 2. Load the index

Loads a previously built index from disk.

```bash
python src/main.py load
```

***

### 3. Print index entries for a word

Displays all documents containing a word and its frequency in each.

```bash
python src/main.py print life
```

Search is **case‑insensitive**:

```bash
python src/main.py print LIFE
```

***

### 4. Find documents matching a query

#### Single‑word search

```bash
python src/main.py find courage
```

#### Multi‑word AND query

```bash
python src/main.py find good friends
```

Returns only pages containing **all** words.

#### Phrase search

```bash
python src/main.py find "good friends"
```

Returns only pages where the words appear **consecutively**, using positional indexing.

#### Empty query handling

```bash
python src/main.py find
```

The system responds with a clear error message.

***

## Ranking and Relevance

Search results are ranked using **TF‑IDF (Term Frequency–Inverse Document Frequency)**:

*   Common words (e.g. *“life”*) may receive zero IDF, resulting in ties
*   Rarer terms produce non‑zero IDF and meaningful ranking
*   Scores are displayed to four decimal places

Execution time for each query is also reported.

    *   Document statistics

*   **Search Engine**
    *   Print command behaviour
    *   Boolean AND queries
    *   Phrase queries
    *   TF‑IDF ranking
    *   Defensive handling of invalid input

Tests are deterministic and do not require live crawling.
